# Particles in an Electric Field

This project simulates the motion of charged particles in a 2D electric field generated by fixed point charges. It was created as part of a vector calculus assignment.

## Description

The simulation builds an electric field using several point charges and shows how particles move under its influence. Each particle responds to the field based on a simple physical model.

The visualization includes:
- Electric field arrows
- Field lines
- A background map of electric potential
- Moving particles with trails
